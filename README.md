## test project for Mindbox company

## Build

```bash
git clone https://github.com/aligadzhiev0595/todoList.git
cd todoList  - $ yarn install || npm i
start script - $ yarn start || npm run start
```

## Project layout

```txt
todoList/
├─ public                    Static files
├─ src/
│ ├─ interfaces              Typescript types
│ ├─ components              React components
│ ├─ redux                   State manager
│ ├─ index.tsx               React app entrypoint
│ ├─ App.tsx                 Root React component
│ └─ index.scss              Root Styles
│
├─ tsconfig                  Typescript config
└─ package.json              NPM dependencies
```
